JSVAuto: Automatically version your script tags!

This web app can help automatically version script tags in your HTML files (as text) so that you can ensure browsers always use the latest version of your files instead of their cached, old ones.


Prior to versioning, all your script tags should have a query string parameter named "autoVer", like this (ignore the bar thing at the end):


Our code increments each version file by 0.0001, but our system accepts any number. If your query string params isn't already formatted, we'll do it for you!
